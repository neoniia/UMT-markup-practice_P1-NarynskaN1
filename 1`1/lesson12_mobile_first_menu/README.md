# Заняття 12 — Адаптив mobile-first та мобільне меню

## Що реалізовано

- Верстка сторінки Flora за принципом mobile-first.
- Адаптив для mobile, tablet і desktop.
- Header з логотипом, навігацією і кнопкою бургер-меню.
- Мобільне меню з класом `is-open`.
- Відкриття та закриття меню через JavaScript.
- Закриття меню по кнопці, по пункту меню, по фону та клавіші Escape.
- CSS-переходи для плавного відкриття меню.

## Основні файли для перевірки

- `frontend/index.html`
- `frontend/css/styles.css`
- `frontend/js/menu.js`

## Як запустити

Відкрити файл `frontend/index.html` через Live Server у VS Code.

## Як перевірити

1. Відкрити сторінку на ширині 375px.
2. Натиснути кнопку меню.
3. Перевірити появу мобільного меню.
4. Закрити меню через хрестик, пункт меню, фон або Escape.
5. Перевірити, що на desktop показується звичайна навігація.

# Flora — Lesson 12

Adaptive Flora page created for the lesson **“Mobile-first adaptive layout. Mobile menu”**.

## What is implemented

- `modern-normalize` is connected through CDN.
- `viewport` meta tag is present in the `<head>`.
- The layout is mobile-first.
- Breakpoints are implemented at `375px`, `768px`, and `1440px`.
- Media queries use `min-width`.
- Horizontal scrolling does not appear from `320px`.
- The mobile menu is fully marked up and styled.
- The menu occupies the whole viewport height.
- The menu is hidden by default.
- Adding the class `is-open` to `.mobile-menu` shows it.
- Opening and closing are handled through JavaScript.
- Images and text content are taken from the Flora mockup.

## Live page

Upload the contents of this folder to the root of the GitHub repository and enable GitHub Pages:

`Settings → Pages → Deploy from a branch → main → /root`

The repository root must contain `index.html`, `css/`, `js/`, `assets/`, `README.md`, and `.nojekyll`.
